================================================================================
                        SPACE SHOOTER GAME
================================================================================

A fast-paced space shooter where you control a ship, avoiding and shooting
down asteroids while trying to achieve the highest score!

================================================================================
HOW TO PLAY
================================================================================

1. Double-click SpaceShooter.exe to start the game
2. Use the controls below to navigate and shoot
3. Avoid asteroids and shoot them to score points
4. The game gets progressively harder as you survive longer
5. Try to beat the high score!

================================================================================
CONTROLS
================================================================================

W or UP ARROW    - Move ship up
A or LEFT ARROW  - Move ship left
S or DOWN ARROW  - Move ship down
D or RIGHT ARROW - Move ship right

SPACEBAR         - Shoot laser

R                - Restart game (after game over)
ESC              - Exit game

================================================================================
GAMEPLAY TIPS
================================================================================

- Stay alert! Asteroids come from the right side of the screen
- Shoot asteroids to destroy them and earn points
- Your score increases the longer you survive
- Each destroyed asteroid gives bonus points
- The game speeds up as you progress - stay focused!
- After game over, enter your name for the high score board

================================================================================
SYSTEM REQUIREMENTS
================================================================================

- Windows 10 or Windows 11 (64-bit)
- 50 MB free disk space
- DirectX 9.0c or later (usually pre-installed)
- Screen resolution: 800x600 or higher

================================================================================
TROUBLESHOOTING
================================================================================

Problem: The game won't start
Solution: Make sure you're running Windows 10/11. Try running as administrator.

Problem: Antivirus blocks the file
Solution: This is a false positive. The game is safe. Add an exception in
          your antivirus software.

Problem: Game runs slowly
Solution: Close other programs. Update your graphics drivers.

================================================================================
ABOUT
================================================================================

This game was created as a Python learning project using pygame.
No installation or Python required - just run the .exe file!

Enjoy the game and good luck beating the high score!
